collections
 Create group cloze. First click on this then select rectangles, the rectangles with changed color will be added to list. Then again click this to generate image files.
view_list
 After downloading svg files and output-note.txt, the details get add this view note sidebar. Click this to view all added notes.
get_app
 On editor window, this button will create questions mask, answers mask and output-note.txt and will download. On View note window, this button will download output-all-notes.txt file combining all previous notes.

Note: Wait for all files to be downloaded. For example, if there are 10 rectangles on screen then total files will be 22 (10 question SVG, 10 Answer SVG, 1 Original SVG, 1 txt file).
live_help
 View help window
add_box
 Add rectangle to editor after importing image to window.
dashboard
 Use this button to create rectangle to any part of screen inside editor area. First click this, then tap on two points (top left corner, bottom right corner) to create rectangle of that width and height. Reset zoom then use this.
delete_forever
 Select a rectangle on editor window and delete that rectangle
zoom_in
 Layout style is set to fixed. So click this to zoom in.
zoom_out
 Layout style is set to fixed. So click this to zoom out.
zoom_out_map
 Reset zoom value. Use it create rectangle at right coordinates.
drag_handle
